# Interactive Taxi-cars Table

Data requested from the database.
Sorting by clicking on the column heading. Search for cars by substring. 

## To Get started


### `git clone https://github.com/EmilioAsteben/taxi-cars-table.git`

### `yarn install`

### `yarn start`
